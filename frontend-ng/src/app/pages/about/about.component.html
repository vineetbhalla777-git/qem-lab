Project report

What we found

Mitigation is not a universal "make it better" button. Here's what the full benchmark sweep actually showed.

Measurement Error Mitigation was the most reliable technique tested — raising GHZ-state fidelity from 0.88–0.96 to 0.98–0.99 at every noise level, since readout error is a purely classical channel that calibration-matrix inversion recovers almost exactly.
Clifford Data Regression delivered consistent 74–86% error reduction on the VQE benchmark across all noise levels tested, degrading gracefully as noise increased.
Zero-Noise Extrapolation actually increased error on our shallow 3-qubit GHZ benchmark at low-to-moderate noise — a real, reproducible limitation on short circuits, not an implementation defect. Mitiq itself warns about this case.
Probabilistic Error Cancellation is highly sensitive to how well its assumed noise model matches the actual device — and to sampling variance at low sample counts.
Virtual Distillation's own ancilla-and-SWAP-gate circuitry introduces noise that can offset the exponential suppression it's designed to provide, especially at shallow depth — a documented effect in the literature, not a bug.

Stack

  • Simulation: Qiskit + Qiskit Aer (noise models, circuit execution)
  • Mitigation: Mitiq (ZNE, PEC, CDR, DDD) + custom implementations (MEM, ancilla-based Virtual Distillation)
  • Backend: FastAPI serving live simulation requests
  • Frontend: Angular (standalone components, Router, HttpClient) + Chart.js
